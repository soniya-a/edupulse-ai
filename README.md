# EduPulse AI

EduPulse AI is an AI-powered smart attendance and student productivity ecosystem designed for educational institutions.

## Features

* QR-based attendance system
* AI attendance prediction
* Subject-wise attendance analytics
* Low attendance alerts
* Smart productivity recommendations
* Teacher dashboard
* Attendance graphs and insights

## Innovation

The system predicts attendance shortages before students become ineligible and provides intelligent productivity recommendations during free periods.

## Technologies Used

* Google AI Studio
* React
* Tailwind CSS
* Firebase
* Vercel
